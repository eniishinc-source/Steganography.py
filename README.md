# Steganography 
#discord.gg/residencescreenshare

Утилита для анализа файлов на возможную стеганографию.

Что делает:

принимает путь к файлу или папке (можно рекурсивно)
анализирует содержимое: сигнатуры файлов, PNG‑чанки, статистику LSB, WAV‑LSB, базовые метаданные
пытается извлечь LSB‑поток (если включено)
выводит в консоль есть ли признаки стего (YES/NO) и причины
сохраняет отчёты в <папка сканирования>\steg_out\...
Что сохраняет:

report.json — полный отчёт
summary.txt — краткий итог
при наличии matplotlib: гистограммы и бит‑плоскости
при PNG: хвост после IEND отдельно
При запуске без аргументов спросит путь и (если папка) предложит сканировать подпапки.
